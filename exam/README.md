# iPAS AI 應用規劃師 — 模擬考網站

離線可用的模擬考系統，直接以瀏覽器開啟 `index.html` 即可（無需伺服器、無需網路）。

## 功能

- **模擬考**（依 5 科分別出題，每題 2 分、滿分 100、70 分及格）
  - 全真模擬：隨機抽 50 題、依正式考試時間倒數（初級 75 分鐘、中級 90 分鐘，可調），交卷後才對答案；時間到自動交卷。
  - 歷屆原卷：選擇梯次，依原始卷面題序作答（初級 3 梯次、中級 2 梯次）。
  - 快速練習：自訂題數、不計時，每題作答後立即顯示官方答案與解析。
- **考後檢討**：成績圈、通過判定、逐題檢討（可篩選 答錯／未作答／已標記／答對／有筆記），標示官方答案、你的作答與解析。
- **每題筆記**：逐題檢討與錯題本中可對任一題新增／編輯筆記（同一題共用一份，存於 localStorage）；清空內容儲存即刪除。
- **錯題本**：答錯自動收錄（含錯誤次數），可依科目「錯題重練」，重練答對自動移除。
- **成績紀錄**：保存於 localStorage（最多 200 筆），可回看任一次的逐題檢討。
- **資料備份**：「成績紀錄」頁可將成績／錯題本／筆記匯出為 JSON，並於其他裝置匯入（合併式：不刪既有資料，重複項目以較新者為準；錯題與筆記僅收錄題庫中存在的題目）。
- **課程教材**：連結 `../學習資源/教材/` 的完整教材與考前重點（相對路徑，整個工作區搬移仍可用）。
- 鍵盤操作：`1`–`4` 或 `A`–`D` 選答、`←`/`→` 換題、`F` 標記。

## 題庫

`data/questions.js`（600 題）由 `學習資源/教材/*_考前重點與試題解析.html` 自動抽取：

| 等級 | 科目 | 題數 |
|------|------|------|
| 初級 | 1 人工智慧基礎概論 | 150（114第四梯次＋115第一次＋115第二次） |
| 初級 | 2 生成式AI應用與規劃 | 150（同上） |
| 中級 | 1 人工智慧技術應用規劃 | 100（114第二梯次＋115第一次） |
| 中級 | 2 大數據處理分析與應用 | 100（同上） |
| 中級 | 3 機器學習技術與應用 | 100（同上） |

官方答案取自 iPAS 公告試題；解析為依官方學習指引撰寫之學習整理，非官方標準解析。

### 更新題庫

教材新增梯次試題後，執行 `python ..\.build\extract_questions.py` 重新產生 `data/questions.js`
（腳本解析教材 HTML 中的 `div.q` 題卡；梯次 ID 對應表在腳本內 `SESSION_NAME`，新增梯次時須補上對應）。
